# GreenPass - App

This repository contains the mobile application of GreenPass. It is realized using Flutter.

## How to run

Clone the project

```bash
git clone https://github.com/GreenPassApp/greenpass-app
```

Go to the project directory

```bash
cd greenpass-app
```

Install dependencies

```bash
flutter pub get
```

Run the app

```bash
flutter run
```
